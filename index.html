O que é um Gabinete

Um gabinete de computador, também conhecido como case, torre ou caixa, é uma estrutura que abriga os componentes principais de um computador pessoal ou servidor. Esse componente é essencial para proteger os elementos internos do computador e fornecer a organização necessária para o funcionamento adequado do sistema.

O gabinete do computador serve principalmente como uma forma de montar fisicamente e conter todos os componentes reais em um computador, como a placa-mãe, disco rígido, unidade óptica, SSD, unidade de disquete, etc. Eles normalmente vêm com uma fonte de alimentação.
